### LuLu Application
Aplicação para controle de medicações.

### Tecnologias
 - Java 15
 - SpringMVC
 - VueJS
 - Bootstrap 5
 - FontAwesome 5
 - Google Fonts

### Como rodar ?
Baixe a versão compilada ou os arquivos fontes para executar.
    
    Compilando
    Requisitos:
    Postgres
    Maven
    Java 15
    
    Execute o comando: mvn clean package
    acesse a pasta target/ e execute o comando: java -jar app.jar
